# Task Manager - Google Sheets Backend

Aplicação de controle de rotina diária com:
- Node.js + TypeScript
- Front-end simples (HTML/CSS/JS)
- Google Sheets como banco de dados
- Apps Script como API ponte (sem Google Cloud / billing)

## Requisitos
- Node 18+
- Conta Google (para o Apps Script)

## Setup
```bash
npm install
cp .env.example .env
npm run dev
